PROGRAM MANAGER

HOW TO RUN:
1. Install XAMPP
2. Start Apache and MySQL
3. Create database called 'program_manager'
4. Run the CREATE TABLE SQL (see setup instructions)
5. Copy files to C:\xampp\htdocs\program-manager
6. Open browser: http://localhost/program-manager

WHAT IT DOES:
- Shows list of all programs from database
- Form to add new programs
- Data is stored in MySQL database

FILES:
- database.php: Connects to MySQL
- index.php: Main page (form + table)
- style.css: Makes it look nice
